# Robot Assistive Arm
Programming information only

## Objectives

### Prototype

Able to perform the following tasks
- Press buttons 
- Pick up items
- Open/close doors

### Code (UI)

- Speed Control
- Threshold
- Motor Combination
- Switching between motors
